Kesimpulan Praktik

pada praktik pembuatan index html ini kita diajarkan untuk membuat halaman beranda sederhana dengan struktur yang lebih rapi dengan menggunakan elemen seperti header,nav, section, article, dan footer. halaman yang kita buat ini memiliki bagian utama yaitu header sebagai judul, navigasi dikiri, konten artikel disisi kanan serta footer dibagian bawah. navigasi berisi beberapa menu yang mengarah ke halaman website multi-halaman. selain itu, penggunaan css didalam style berfungsi untuk mengatir tampilam seperti layout, warna, ukuran, dan posisi elemen. dengan menerapkan responsive design menggunakan @media, sehingga tampilan akan menyesuaikan ketika dibuka dilayar kecil (seperti HP).
