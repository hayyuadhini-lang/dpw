Penjelasan AJAX dan jquery

1. Pengertian AJAX

AJAX (Asynchronous JavaScript and XML) adalah teknik dalam pengembangan web yang memungkinkan haaman web mengambil atau mengirim data ke server tanpa harus memuat ulang seluruh halaman. Dengan AJAX, data dapat diperbarui secara Asynchronous sehingga website menjadi lebih cepat dan interaktif.

2. Pengertian jQuery

jQuery adalah library JavaScript yang digunakan untuk mempermudah penulisan kode JavaScript. jQuery membantu developer melakukan manipulasi HTML,mengatur event, membuat animasi, dan Menggunakan AJAX dengan lebih sederhana dan singkat.

3. Cara Menggunakan jQuery

untuk menggunakan jQuery, kita harus menambahkan library jQuery ke dalam file HTML. contohnya menggunakan CDN seperti berikut:

<script src="https://code.jquery.com/jquery-3.6.0.min.js"</script>
