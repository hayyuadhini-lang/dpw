ini adalah elemen pertama menggunakan DIV
ini DIV di dalam DIV namun membuat elemen blok baru. ini SPAN di dalam, DIV tidak membuat elemen blok baru

pada prraktik kali ini kita belajar tentang perbedaan antara element block level seperti div dan elemen inline seperti span. elemen div itu membuat blok baru yang memenuhi lebar tertentu, sehingga konten didalamnya tersusun dalam kotak terpisah. sedangkan, span itu tidak membuat blok baru. di praktik ini kita menggunakan display: blok; yang tampilannya seperti elemen blok. pada praktikum ini kita ditujukkan bahwa CSS dapat mengubah perilaku default elemen HTML, jasi kita bisa mengatur apakah element tampil sebagai blok atu inline sesuai kebutuhan desain.
