Arahkan mouse ke sini!

pada praktik kali ini kita diajarkan untuk membuat kode link dengan pseudo-class, jadi kita belajar bagaimana CSS dapat mengatur tampilan link sesuai kondisi interaksi pengguna. pseudo link itu seperti :link, :visited, :hover, dan :action yang memberikan gaya berbeda pada setiap keadaan, misalnya saat link belum dikunjungi, sudah dikunjungi, diarahkan dengan mouse atau sedang diklik. praktikum kali ini menunjukkan pada kita bahwa CSS tidak hanya mengatur elemen secara statis, tetapi juga mampu mengtur perilaku dinamis berdasarkan inrtraksi pengguna.
